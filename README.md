**ChatSync** is a Chrome extension that turns AI-generated plans (ChatGPT, Claude, Gemini, etc.) into real calendar schedules.
---

## ✨ Features

- One-click import directly from ChatGPT  
- Smart parsing of weekdays 
- Time recognition  
- Weekly repeating events  
- Clean preview of upcoming schedule  
- Downloadable `.ics` calendar files  
---
---

##  Futures

- Auto-detect Claude/Gemini/Perplexity  
- Week-range parsing (e.g., "Week 1–4")  
- Google Calendar direct sync (OAuth)  
- Task mode (non-date plans become todo lists)
